简述：
    本套框架是适用于bees的一套pytest框架，可以实现读取excel表格数据自动完成接口测试，并对结果进行断言，然后生成html的测试报告。
    pytest框架是基于python语言编写，版本为Python3及以上，通过Pycharm进行编译，简单灵活，容易上手。
    首先通过使用pytest.mark.parametrize方法进行参数读取，然后使用requests第三方库实现对接口的操作，自后通过allure库生成美观的测试报告。
前置条件：
    在使用框架之前，需安装Python3及以上版本，Pycharm编译器。通过github获取项目地址，执行安装配置文件pip install -r requirements.txt安装所有依赖包
框架结构：
    beesService     ==>     执行测试用例目录
    businessInfo    ==>     业务流程脚本目录
    caseinfo        ==>     存放测试用例excel目录
    commons         ==>     存放工具类目录
    config          ==>     存放配置文件目录
    projectService  ==>     项目维度执行目录
    projectBusiness ==>     业务流程维度执行目录
    files           ==>     存放文件上传下载目录
    logs            ==>     存放日志目录
    mpServices      ==>     存放各服务执行测试用例目录
    reports         ==>     存放报告目录
    temps           ==>     存放缓存目录
    run.py          ==>     执行所有测试用例文件
    Pytest.ini      ==>     pytest执行规则文件
    explain.txt     ==>     框架描述及使用说明
